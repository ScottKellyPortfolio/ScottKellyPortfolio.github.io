Projects

Weight And Balance Calculator

Design, Development

The idea for this application came from my father, who pilots private charter jets. The purpose of the app, is that it makes it simple to calculate the weight and balance of an aircraft before it flies. It uses built in information for different aircraft to make it simple to give your weights as inputs, and it will tell you if the aircraft is safe to fly for the weights the user input. At this point in time, it is still a work in progress, but is close to being finished.

The Challenge

Simplify calculating the various weights of small aircraft to make calculating the weight and balance easier. Make it easy to use and make it mobile friendly so as to avoid needing a desktop or laptop.

The Solution

A responsive web application that can be used on mobile devices to increase ease and productivity. Replaces the need to use a computer or desktop application.

The Result

Still to come! This is a work in progress project as I am the only person working on it. Unfortunately because it is a personal project, it takes the backseat for more pressing issues.

A.I.M.S.

Design, Development

A.I.M.S. otherwise known as the Apprentice Intership ManageMent System, is an application built for The Eleven Fifty Academy. The Challenge

Create something that makes managing courses and people easier. Used by Eleven Fifty, but will be scalable to use with other companies.

The Solution

A responsive web application written in C# and ASP.Net framework along with native mobile apps that help organize and manage people.

The Result

Still a work in progress. Our first iteration includes the ability to make users and an admin login and portal.
